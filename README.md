## :open_book: Description

## :whale: Docker

```bash
$ npm install
```

## :cd: Installation


```bash
# Up database
$ docker-compose up -d

# down database
$ docker-compose down
```

## :checkered_flag: Running the App

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## :mag_right:  Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
